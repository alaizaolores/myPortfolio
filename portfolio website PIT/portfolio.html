Car Rental System

This was our Performance Innovative Task (PIT) in Object-Oriented Programming during the first semester of our second year. We were assigned to create a Car Rental System using a GUI in the Java programming language. The Car Rental System project is a comprehensive application designed to manage the rental process for a car rental service. The system aims to streamline the workflow from customer registration to vehicle returns. This project emphasizes the practical application of Object-Oriented Programming (OOP) principles, GUI development, and database integration.
